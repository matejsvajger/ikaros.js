## Installation

Prerequisites: [Node.js](https://nodejs.org/) 8+, **npm** 3+.

    npm install --save ikaros

## Usage

### ES5

    var Ikaros = require('ikaros').default;

### ES2015+

    import Ikaros from 'ikaros';
